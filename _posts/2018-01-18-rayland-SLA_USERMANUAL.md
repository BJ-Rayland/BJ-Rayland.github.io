`RAYLAND-SLA`是乐轩锐蓝推出的**集振镜控制**和**电机控制**于一体的控制板。性能可以对标ScanLab的同类型板卡。



![1B9E6A5E-FA15-4CD4-9C86-285A7B57FAF6]({{site.baseurl}}/images/1B9E6A5E-FA15-4CD4-9C86-285A7B57FAF6.png)

## 基本参数

| 尺寸        | 115×93 mm    |
| --------- | ------------ |
| 重量        |              |
| 工作温度      | -20～65℃      |
| 主处理器主频    | 1.4GHz       |
| 主处理器类型    | 4核 Cortex A7 |
| 协处理器      | 72MHz        |
| 内存容量      | 1Gbytes      |
| Flash     | 4Gbytes      |
| 最大外接Flash | 32Gbytes     |
